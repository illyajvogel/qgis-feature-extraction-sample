# QGIS Feature Extraction and Cartographic Layout Sample

This repository contains a GIS work sample created in QGIS from high-resolution aerial imagery.

## Overview
This project demonstrates manual feature extraction, vector layer creation, and cartographic layout design. Starting from imagery alone, I identified and digitized site features including buildings, roads, parking areas, vegetation, pedestrian paths, utilities, signage, and other infrastructure elements.

## Skills Demonstrated
- Manual feature extraction from imagery
- Vector data creation and organization
- Layer classification and symbology
- Cartographic layout design
- Spatial interpretation and visual communication
